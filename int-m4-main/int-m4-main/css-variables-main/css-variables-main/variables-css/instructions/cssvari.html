Pourquoi utiliser des variables en CSS

Les sites peuvent avoir des feuilles de style ou de nombreuses valeurs sont répétées (couleurs, typos...) qu'il faudra mettre à jour si besoin. Les variables CSS permettent de stocker une valeur à un endroit plus de réutiliser cette valeur (on factorise ainsi le code).
